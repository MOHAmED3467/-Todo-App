# ✅ Todo App

A simple and stylish Todo List app built using **HTML, CSS, and JavaScript**.  
It allows you to:
- Add new tasks
- Mark tasks as done
- Delete tasks
- Save tasks in **LocalStorage** (tasks stay after refreshing the page)
